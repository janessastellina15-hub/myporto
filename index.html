About Me

Saya Janessa Stellina Yedyra, atau biasa dipanggil Jane. Saya memiliki ketertarikan dalam bidang Desain Grafis, Fotografi, dan Vidiografi. Saya senang belajar hal-hal baru dan terus berusaha mengembangkan kemampuan saya di bidang kreatif. Software





CorelDRAW

CorelDRAW(CDR) adalah software desain grafis vektor yang digunakan untuk membuat dan mengedit gambar berbasis vektor. Saya cukup memahami fitur yang ada di dalamnya, mulai dari pengaturan layout, membuat vektor, sampai memoles detail desain.

Adobe Ilustrator

Adobe Illustrator(AI) adalah software desain grafis berbasis vektor yang digunakan untuk membuat logo, ikon, ilustrasi digital, dll. Saya sudah cukup memahami dasar-dasarnya, meski belum sepenuhnya menguasai seluruh fiturnya.

Adobe Photoshop

Adobe Photoshop(PS) adalah software editing gambar yang digunakan untuk mengolah foto, manipulasi visual, dan memperbaiki detail. Saya sudah cukup memahami dasar-dasarnya, meski belum sepenuhnya menguasai seluruh fiturnya.

Canva

Canva adalah platform desain grafis yang mudah digunakan. saya cukup memahami banyak fitur di dalamnya, namun terkadang masih belum mampu menghasilkan desain yang benar-benar menarik. Meski begitu, saya tetap terus belajar.

VS Code

Visual Studio Code(VS Code) adalah editor kode yang bisa diperluas dengan berbagai ekstensi untuk mempermudah proses coding. Saya cukup memahami fiturnya, dan portofolio yang saya buat juga dikerjakan menggunakan VS Code.

Figma

Figma adalah software desain grafis yang digunakan untuk membuat desain visual, seperti tata letak situs web/elemen aplikasi. Saya cukup paham mengenai fiturnya, dan CV yang saya buat juga dikerjakan menggunakan Figma.
